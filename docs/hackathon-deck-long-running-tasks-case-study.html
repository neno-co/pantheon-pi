Pillar 1 · define done

Pillar 1: The Truth Conditions Framework

/define-project interviews the team and produces Truth Conditions: the north star and the loop's stop condition.
Each is [auto] (an agent runs a check and asserts the outcome) or [manual] (an explicit human sign-off gate).
Project, conditions & milestones are written straight into Linear as the shared source of truth.
Truth Conditions · real example (Reconciliation KPIs)
[auto]Every bill/receipt lifecycle timestamp + actor is queryable in Postgres: created, first review, approval, accepted link, reconciled (from Exact sync).
[auto]Processing, queue, review & reconciliation elapsed times derive from SQL alone: no logs, UI reconstruction, or spreadsheets.
[auto]No partial KPIs: an approved bill has both approvedAt + approvedByUserId; a failed Exact sync never counts as reconciled.
[auto]Approval backfills firstReviewedAt / ById so the created → first-review → approval funnel stays monotonic.
[manual]The Reconciliation KPIs skill documents the query path, SQL definitions, cutoff rules & null semantics, including safe GCP read-only access.
[manual]Nick / Matthew / Yaroslav can review infographics tying the KPI lifecycle to accountant-time and clients-per-accountant goals.

Get started · long-running loops, the Neno way

The biggest lift is building trust with agents before you let them run overnight.

$ claude auth login # Claude / Anthropic auth
$ curl -fsSL https://pantheon.viche.ai/install.sh | bash
   # clones repo, installs deps, registers the Pi package, runs init
$ pantheon # launch Pi with Athena as default primary
# then, inside the pantheon session:
› /login # authenticate openai-codex + gemini providers
› /define-project # pin truth conditions into Linear
› /goal "implement [Linear project] until Dike approves the truth conditions"
Requires Bun, Pi, acpx & Git on your PATH.
It works as a foundation as long as you have:
Enforced pre-commit hooks: tests, linting, integration & DB-schema verification
Consistent CI/CD pipelines to catch errors at deploy time
Humans at the gates that matter
Don't trust the agent. Trust the loop.
